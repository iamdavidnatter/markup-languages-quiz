# Markup Languages Interaktives Quiz V2

Statische Web-App mit Multiple-Choice-Fragen aus:
- Wiederholung Markup Languages
- Moodle-Wissenschecks
- Hauptklausur Markup Languages 2025

## Lokal testen
`index.html` im Browser öffnen.

## Render Deployment
1. Repository auf GitHub erstellen.
2. Alle Dateien in dieses Repository hochladen.
3. Render → New → Static Site.
4. Repository verbinden.
5. Build Command leer lassen.
6. Publish Directory: `.`
